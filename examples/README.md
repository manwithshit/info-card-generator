# 示例卡片

这个目录包含了一些示例卡片，展示不同的设计风格。

## 风格示例

### 1. Blueprint 蓝图风格
- 文件：`example_blueprint_style.html`
- 适用：技术架构、系统设计、工作流介绍
- 特点：深蓝背景、网格线、青色高亮、工程图风格

### 2. Futuristic 未来极简风格
- 文件：`example_futuristic_style.html`
- 适用：趋势分析、产品展望、科技报告
- 特点：深色背景、玻璃拟态、渐变光晕、现代感强

## 如何使用

1. 运行截图命令生成 PNG：
   ```bash
   node scripts/capture_card.js examples/example_blueprint_style.html
   ```

2. 或者直接在浏览器中打开 HTML 文件预览效果。

## 创建自己的风格

1. 复制 `templates/basic_card.html` 作为起点
2. 修改 CSS 变量和样式
3. 替换内容
4. 运行截图脚本
